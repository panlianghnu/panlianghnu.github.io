声明式渲染

Vue 应用会将其挂载到一个 DOM 元素(比如DIV) 上，然后对其完全控制

结果显示 Hello Vue！

条件渲染

v-if 条件渲染元素，可以配合 v-else-if、v-else 使用

v-show 和 v-if 基本一样，都能控制元素是否显示

v-if 是真正的条件渲染，是惰性的。而 v-show 是始终渲染，只简单的 CSS 切换 display

v-if 的切换开销更大，初始化开销更小

循环渲染/事件
